Thank you for using web-setup

This is a very simple cli to setup html, css, and js with jQuery and Bootstrap to help coders get started easily.

The idea is to just create the folders, download bootstrap and jquery and write a simple gulp js that can process the html, css and js at a script command.

Dependencies

  • Bootstrap (for layout)
  • jQuery (not used but included for convenience)

Do note that this is a very simple web project setup. You may need to use a more than just jQuery and Bootstrap to create a webpage.

You may fork the project to customize the project based on your need. For queries and ideas on how to customize, you can also find me kenhua-l@github.
